About Us

  StaVia is an automated end-to-end trajectory inference (TI) framework that uncovers cellular trajectories permeating large-scale single-cell spatial and temporal atlases without sacrificing fine-grained details. It leverages Lazy-teleporting random walks (LTRW) with memory to accurately pinpoint end-to-end trajectories in the atlas. StaVia’s memory feature is like an alert GPS that not only focuses on the road immediately ahead but also keeps track of the overall journey. It remembers where each cell has been and where it could be headed, making it less likely to miss critical turns (or transition states), and providing a more complete picture of the cell’s developmental journey.


  The single-cell data is first represented by a scKNN graph, where additional edges between cells in adjacent sequential groups are added for temporal data, and the input gene expression is transformed into weighted average of a cell’s own cells and spatial neighbors. Subsequently, A cluster graph is constructed where nodes are PARC-based clusters of single cells, and lineage pathways are computed using second-order LTRWs with memory. An edge-bundled cluster graph is then constructed via kernel density estimation (KDE) which transforms the graph into a density map. Finally, A single-cell embedding is constructed for Atlas View based on the layout of the forward-biased TI cluster graph. A complete Atlas view can be generated by clustering the single-cell 2-D embedding layout with kmeans and KDE.

Tools Offered

Time-Series

StaVia leverages higher order random walks with memory to highlight key end-to-end differentiation pathways

Spatial-Temporal

Using sequential metadata (i.e., temporal labels) and spatial tissue coordinates to guide the cartography

Atlas Mapping

Via 2.0 offers sophisticated visualization tools aimed to capture connectivity and cellular resolution simultaneously.

RNA Velocity

StaVia’s framework is flexibly compatible with diverse input data: RNA velocity, spatial coordinates, and temporal information
